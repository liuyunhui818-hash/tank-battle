# 坦克大战 (Tank Battle)

经典坦克大战游戏，使用 HTML5 Canvas 和原生 JavaScript 实现。

## 操作方式

| 按键 | 功能 |
|------|------|
| WASD / 方向键 | 移动坦克 |
| 空格键 / J | 发射子弹 |
| P | 暂停/继续 |

## 游戏特色

- 3 个不同关卡，难度递增
- 砖墙可被子弹摧毁，钢铁墙壁不可摧毁
- 保护基地（菱形标志），基地被摧毁则游戏结束
- 3 条生命，击败敌方坦克获取分数
- 粒子特效和屏幕震动

## 运行方式

直接在浏览器中打开 `index.html` 即可开始游戏。
